# Setup Guide

## Quick Start

1. **Install Node.js** (if not already installed)
   - Download from [nodejs.org](https://nodejs.org/)
   - Version 18 or higher recommended

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   - Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` - Next.js app directory
  - `components/` - React components
  - `page.tsx` - Main page
  - `layout.tsx` - Root layout
- `docs/` - Sample markdown files (optional, currently using localStorage)
- `prototypes/` - Sample code files (optional, currently using localStorage)

## Features

- **Docs Section**: Create and edit markdown documents
- **Prototypes Section**: Store React code snippets
- **Dark Mode**: Toggle theme in the header
- **Local Storage**: All data persists in your browser

## Next Steps
